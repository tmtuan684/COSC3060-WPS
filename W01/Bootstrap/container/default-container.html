Breakpoints

Breakpoints are predefined width before which element's width is 100% and after which the element's width is a predefined max-width.

.container sets max-width at each responsive breakpoint.
100% wide until small breakpoint where screen size is 540px
100% wide until medium breakpoint where screen size is 720px
100% wide until large breakpoint where screen size is 960px
100% wide until extra large breakpoint where screen size is 1140px
100% wide until extra extra large breakpoint where screen size 1320pxs
.container-fluid sets width to 100% at all breakpoints.
